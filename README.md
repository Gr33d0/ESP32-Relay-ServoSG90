# ESP32-Relay-ServoSG90
ESP32 activate Relay that activate servo

List of items:

Esp32-TTGO
Servo SG90
Relay 5v
External battery (5V)

Board: LilyGo T-Display

Connections:

Relay:

DC+ - 5V (Board)
DC- - GND (Board)
IN - Pin 17 (Board)
NO - VCC (External Battery)

Servo:

Brown - GND (External battery)
Red - COM (Relay)
Orange - Pin 16(Board)
