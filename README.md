# ESP32-Relay-ServoSG90
ESP32 activate Relay that activate servo

List of items:<br />

Esp32-TTGO<br />
Servo SG90<br />
Relay 5v<br />
External battery (5V)<br />

Board: LilyGo T-Display<br />

Connections:<br />

Relay:<br />
<br />
DC+ - 5V (Board) <br />
DC- - GND (Board)<br />
IN - Pin 17 (Board)<br />
NO - VCC (External Battery)<br />

Servo:<br />

Brown - GND (External battery)<br />
Red - COM (Relay)<br />
Orange - Pin 16(Board)<br />
